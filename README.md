# Gipad
Gipad – With AI powered Notepad.


Gipad is a modern, cyberpunk-inspired notepad built with Python using CustomTkinter.
It combines a sleek dark UI with powerful editing features and a lightweight AI assistant powered by Wikipedia.

# ✨ Features



# 💾 File Management
New / Open / Save / Save As
UTF-8 file support
Smart status updates
# 🎨 Customization
Font selection
Text color changer
Background color changer
Clean cyberpunk dark theme
# ⚡ Productivity
Undo / Redo support
Keyboard shortcuts:
Ctrl + N → New tab
Ctrl + O → Open file
Ctrl + S → Save file
# 🤖 Built-in AI Assistant
Ask questions directly inside the editor
Fetches intelligent summaries using Wikipedia API
Inserts results directly into your notes
# 🖤 Modern UI
Built with CustomTkinter
Smooth dark interface
Minimal sidebar design
Developer-friendly layout

# 🚀 Installation
pip install customtkinter requests
# ▶️ Run
python gipad.py
📦 Build EXE
python -m pip install pyinstaller
python -m PyInstaller --onefile --noconsole gipad.py

or,,
C:/Users/USER/AppData/Local/Microsoft/WindowsApps/python3.11.exe -m PyInstaller --onefile --windowed gipad.py
